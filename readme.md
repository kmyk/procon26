## ビームサーチ

ビームサーチです

-   予選 1位
-   朝のOB戦 2位
-   準決勝 2位
-   決勝 4位
-   エキシビションマッチ 6位

## 補助ツール

SDL2.0,SDL2_image,SDL2_gfx,SDL2_ttfが必要です

``` sh
    $ cd GUIapp
    $ make
    $ ./GUIapp.elf <game num>
```
![画面](GUIapp/doc/img/image1.png "画面")
-Tabキー: 裏返す
-Spaceキー: 90度回転
